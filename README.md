# power-bi-sales-performance-dashboard

## Projekt célja
Power BI-ban Power Query segítségével adattisztítás betöltés előtt, utána riport elkészítése amely elemzi egy Superstore teljesítményét, profitot DAX mérések és dashboard használatával.

## Oldalak
- Sales Overview (KPI kártyák, régió szerinti teljesítmény, ügyfél típus szerinti profit)
- Product Performance (alkategória szerinti mérések)
- Profitability (top államok szerinti profit, szállítás típus szerinti profit)

## Fő kimutatások
- Egyes államok kiemelkedő profitot termelnek
- Magasabb kedvezmények alacsonyabb profitot hoznak több esetben
- Némely termék sokkal több profitot termel a kedvezményéhez képest
