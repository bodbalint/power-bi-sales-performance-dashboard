## power-bi-sales-performance-dashboard

## Project goal
In Power BI before loading a datatable, cleaning data with Power Query, then making a report which analyses a Superstore's performance and profit using DAX measures and a dashboard for visualization.

## Pages
- Sales overview (KIP cards, performance by region, profit by customer type)
- Product Performance (Measures by Sub-Categories)
- Profitability (Profits in the top states, profit by delivery type)

## Main insights
- Some states show marginally better profits
- The higher the discount, the less profit it generates in most cases
- Some products generate a huge profit compared to its discount
